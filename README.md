## 💡 주요 학습 내용

### 기술 스택

- **Testing**: JUnit 5, Testcontainers, MockMvc

## 🛠️ 설치 및 사용법

### 필요한 도구
```bash
# Git 설정
git config --global user.name "Your Name"
git config --global user.email "you@example.com"

# Markdown 도구 (로컬 설치)
npm install

# GitHub CLI
brew install gh

# SDKMAN (Java 버전 관리)
curl -s "https://get.sdkman.io" | bash
```

### 문서 검증
```bash
# 링크 검사
npm run lint:links

# 포맷팅
npm run format

# 문법 검사
npm run lint:md
```
